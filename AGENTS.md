# 苏州园林数据可视化项目

## 项目目标

我希望通过这个项目，实现以下目标：

- 数据大盘：展示苏州园林的各类数据指标，帮助用户了解整体情况
- 地图可视化：通过地图展示苏州园林的地理分布和相关数据
- 信息展示：展示苏州园林的简要介绍和图片，提升用户体验
- 交互功能：提供筛选、搜索等交互功能，方便用户查找感兴趣的内容

总的来说，该项目不仅需要能够展示数据，更能帮助用户理解和发掘数据，体现数据中的 Insights。

## 目录约定

- ./dataset: 苏州园林数据和图片
- ./archived: 旧数据，禁止使用
- ./archived/VIS: 旧版可视化项目，可供参考（索引文档：docs/OLD_VIS_INDEX.md）

唯一的真实数据源头：dataset/SuzhouGardenListFull.csv 和 dataset/images 中的图片（以园林名称命名的文件夹）

## 工程指引

- 利用优秀的前端现成组件、成熟的 NPM 包，提升开发效率
- 组件化开发，提升代码复用性
- 保持代码简洁，避免冗余，重构时需要删除已经不再使用的代码
